# Massive Tanks: Armored Command

A strategic base-building and warfare game designed as a companion "meta-game" for **World of Tanks** players.

## Infrastruktur

| Komponente | Details |
|-----------|---------|
| **Sprache** | TypeScript, JavaScript |
| **Frontend** | React (via Vite) |
| **Backend** | Node.js |
| **Status** | Aktive Entwicklung |

## Über das Spiel

Massive Tanks erweitert das World-of-Tanks-Erlebnis um eine strategische Meta-Ebene mit Basisbau, Ressourcenmanagement und taktischen Entscheidungen.
