How to Run Massive Tanks: Armored Command

This document provides instructions for running the Massive Tanks: Armored Command prototype.

Prerequisites

To run the application, you need:

  • A modern web browser (Chrome, Firefox, Edge, or Safari)
  • Internet connection (for loading external resources like Font Awesome)

Running the Application

There are several ways to run the application:

Method 1: Open the HTML File Directly

  1. Navigate to the massive-tanks/client directory
  2. Double-click on index.html to open it in your default web browser

Note: Opening the file directly may have some limitations due to browser security restrictions. If you encounter issues with loading resources or API calls, try one of the other methods below.

Method 2: Using a Local Web Server

For the best experience, we recommend running the application using a local web server:

Using Python's built-in HTTP server:

  1. Open a terminal or command prompt
  2. Navigate to the massive-tanks/client directory
  3. Run one of the following commands based on your Python version: 
    # For Python 3
python -m http.server 8080

# For Python 2
python -m SimpleHTTPServer 8080
  4. Open your web browser and navigate to http://localhost:8080

Using Node.js and http-server:

  1. If you don't have http-server installed, install it globally: 
    npm install -g http-server
  2. Navigate to the massive-tanks/client directory
  3. Run: 
    http-server -p 8080
  4. Open your web browser and navigate to http://localhost:8080

Using Visual Studio Code Live Server extension:

  1. Open the project in Visual Studio Code
  2. Install the "Live Server" extension if you don't have it already
  3. Right-click on index.html and select "Open with Live Server"
  4. The application will open in your default browser

Using the Application

Once the application is running:

  1. Enter any username on the login screen (password is not validated in the prototype)
  2. Click "Login" to enter the application
  3. Navigate between different views using the navigation bar
  4. In the Base view, select a building type and click on the grid to place it
  5. In the Garage view, click "Import Tanks" to simulate importing tanks from World of Tanks
  6. In the Missions view, click "New Mission" to create a strategic mission

Note: This is a prototype version with simulated functionality. No actual connection to the Wargaming API is made.

Troubleshooting

Console Errors

If you encounter issues, open your browser's developer console (F12 or Ctrl+Shift+I in most browsers) to check for error messages.

Common Issues

  • CORS errors: If you see Cross-Origin Resource Sharing (CORS) errors in the console, you need to use a local web server as described in Method 2.
  • Missing resources: Ensure you have an internet connection for loading external resources like Font Awesome.
  • JavaScript disabled: Make sure JavaScript is enabled in your browser.

Resetting Data

If you want to reset all application data:

  1. Open your browser's developer console (F12 or Ctrl+Shift+I)
  2. Type App.resetAllData() and press Enter
  3. Confirm the reset when prompted
