Running Massive Tanks: Armored Command

This guide will help you run the Massive Tanks: Armored Command prototype on your local machine.

Note: This is a prototype version designed to demonstrate the concept and UI. The full version will include complete integration with the Wargaming API and more features.

Method 1: Using a local web server

The recommended way to run Massive Tanks is to use a local web server. This ensures all features work correctly.

Option A: Using Node.js and http-server

  1. Install Node.js if you don't have it already.
    • Download from https://nodejs.org/
    • Follow the installation instructions for your operating system
  2. Install http-server globally: 
    npm install -g http-server
  3. Navigate to the project directory in your terminal or command prompt: 
    cd path/to/massive-tanks/client
  4. Start the server: 
    http-server
  5. Open your browser and navigate to: 
    http://localhost:8080

Option B: Using Python's built-in HTTP server

  1. Make sure Python is installed on your system.
    • Most macOS and Linux systems have Python pre-installed
    • Windows users can download from python.org
  2. Navigate to the project directory in your terminal or command prompt: 
    cd path/to/massive-tanks/client
  3. Start the server:
    • For Python 3.x: 
      python -m http.server
    • For Python 2.x: 
      python -m SimpleHTTPServer
  4. Open your browser and navigate to: 
    http://localhost:8000

Method 2: Opening the HTML file directly

You can also open the HTML file directly in your browser, but some features might not work correctly due to browser security restrictions.

  1. Navigate to the massive-tanks/client directory
  2. Double-click on index.html to open it in your browser
Warning: When opening the file directly, localStorage might not work correctly in some browsers. This means your game progress might not be saved between sessions.

Using the Application

  1. Login: Use any username to log in (no password required in the prototype)
  2. Link WoT Account: Click "Link World of Tanks Account" to simulate connecting to the Wargaming API
  3. Navigate: Use the navigation bar to switch between views (Base, Garage, Missions, etc.)
  4. Build your base: Select a building type and place it on the grid
  5. Import tanks: In the Garage view, click "Import Tanks" to get sample tanks
  6. Create missions: In the Missions view, click "New Mission" to create a strategic mission

Troubleshooting

Console Errors

If you encounter any issues, check your browser's developer console (F12 or Ctrl+Shift+I on most browsers) for error messages.

LocalStorage Issues

If your game progress isn't saving, make sure:

  • You're running the app through a web server (Method 1)
  • You haven't blocked cookies or localStorage in your browser settings
  • You're not using private/incognito browsing mode

Display Issues

If the UI doesn't display correctly:

  • Make sure you're using a modern browser (Chrome, Firefox, Edge, Safari)
  • Check if all files are properly loaded in the Network tab of developer tools
  • Try clearing your browser cache
